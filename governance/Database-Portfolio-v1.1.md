# Multi-Database Search Portfolio — v1.1
**Issued:** 2026-03-18 | **Supersedes:** Informal Tier 1-4 list

## Tier 1 — Always Required

| Database | Coverage | Access |
|---|---|---|
| PubMed / MEDLINE | Biomedical from 1946; 35M+ records | Free; MCP available |
| Embase | European + international; drug/conference strength | Ovid or Embase.com; surrogate if unavailable |

## Tier 2 — Required for Clinical Reviews

| Database | When |
|---|---|
| CENTRAL (Cochrane) | Any RCT arm present |
| Web of Science (SCI) | Citation tracking of key included articles |

## Tier 3 — Domain-Specific

| Database | When |
|---|---|
| CINAHL | Nursing / allied health outcomes |
| PsycINFO | Mental health or QoL outcomes |
| Scopus | When WoS unavailable |
| LILACS | Latin American / Caribbean populations |

## Tier 4 — Supplementary

| Database | When |
|---|---|
| Scholar Gateway | Always — semantic search catches Boolean gaps |
| Grey literature | Cochrane-standard or guideline-informing reviews |

## Minimum Portfolio by Review Type

| Review type | Minimum set |
|---|---|
| Prevalence / incidence meta-analysis | PubMed + Embase/surrogate + Scholar Gateway + manual |
| Intervention systematic review | PubMed + Embase + CENTRAL + Web of Science |
| Scoping review | PubMed + Embase + Scholar Gateway + CINAHL |
| Mendelian randomisation | PubMed + OpenGWAS + GTEx + manual |

## Embase Surrogate Protocol

When Embase direct access is unavailable:
1. Four Emtree-mapped PubMed search strings covering all PICO components
2. Scholar Gateway semantic search (minimum two queries)
3. Disclose in Methods, PRISMA item 6, and PROSPERO registration
