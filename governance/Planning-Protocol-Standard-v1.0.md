# Planning Protocol Standard — v1.0
**Issued:** 2026-03-18 | **Authority:** Julian Borges, MD | **Status:** ACTIVE

## Governing Principle

Written planning is mandatory before any systematic review search. PROSPERO/OSF registration
is strongly recommended as the public signal of that planning but is not itself the hard
constraint. The planning document is the audit trail.

## Mandatory Planning Fields

| Field | Content | Framework |
|---|---|---|
| Review title | Working title: population + exposure + outcome | — |
| PICO | Population, Intervention/Exposure, Comparator, Outcome | PICO |
| Research question | Single structured question | FINER |
| Primary outcome | One pre-specified | — |
| Secondary outcomes | Ranked list | — |
| Eligibility criteria | Inclusion and exclusion by design/population/outcome | — |
| Search strategy | Databases, Boolean string, date/language restrictions | PROSPERO Item 6 |
| Synthesis method | Narrative vs quantitative; model; effect measure | — |
| Sensitivity analyses | Pre-specified exclusion scenarios | PRISMA 13f |
| Risk of bias tool | Named tool and domains | GRADE |

## PROSPERO/OSF Registration Guidance

| Scenario | Recommendation |
|---|---|
| Intervention review | Register PROSPERO before search — strongly recommended |
| Prevalence/incidence meta-analysis | Register OSF before search (PROSPERO prevalence not yet live) |
| Rapid review | Complete planning; register PROSPERO retrospectively with disclosure |
| Searches already executed | Register retrospectively; disclose in Methods, PRISMA, Limitations |
